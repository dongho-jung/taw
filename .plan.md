# Plan: Expand Shell Popup to Full Window Width

## 현재 상황
- Alt+P로 shell popup을 띄우면 현재 pane 아래에 40% 크기로 split됨
- 2개 pane이 vertical split 상태일 때, 새 shell pane이 현재 pane만 split해서 전체 창 너비를 차지하지 않음

## 목표
- Shell popup이 전체 window 아래에 full-width로 표시되도록 변경
- 기존 pane들은 위로 올라가고, shell pane은 아래에 전체 너비로 표시

## 해결 방법
tmux `split-window` 명령어의 `-f` (full) 플래그를 사용:
- `-f`: 현재 pane이 아닌 전체 window를 기준으로 split
- `-v -f`: 전체 window 너비로 아래에 pane 생성

## 작업 단계

### 1. `SplitOpts` 구조체에 `Full` 필드 추가
**파일**: `internal/tmux/client.go` (line 102-110)
```go
type SplitOpts struct {
    Target     string
    Horizontal bool
    Size       string
    StartDir   string
    Command    string
    Before     bool
    Full       bool   // -f flag: full-width/height split
}
```

### 2. `SplitWindowPane` 함수에 `-f` 플래그 처리 추가
**파일**: `internal/tmux/client.go` (line 320-350)
```go
if opts.Full {
    args = append(args, "-f")
}
```

### 3. popup-shell 명령어에서 `Full: true` 옵션 사용
**파일**: `cmd/taw/internal.go` (line 867-872)
```go
newPaneID, err := tm.SplitWindowPane(tmux.SplitOpts{
    Horizontal: false,
    Size:       "40%",
    StartDir:   panePath,
    Command:    shellCmd,
    Full:       true,  // 전체 window 기준으로 split
})
```
