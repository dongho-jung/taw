# PAW Agent Instructions

When the user asks about **PAW-specific operations**, read `.paw/HELP-FOR-PAW.md` for detailed guidance.

## Trigger Keywords

Read the help file when user mentions any of:
- **Hooks**: `pre_worktree_hook`, `post_task_hook`, `pre_task_hook`, `post_merge_hook`, `pre_merge_hook`
- **Config**: `.paw/config`, PAW hooks, log settings
- **Logs**: PAW logs, task logs, log viewer
- **PAW operations**: worktree behavior, task options, PAW features

## Quick Reference

| Topic | Action |
|-------|--------|
| Configure hooks | Edit `.paw/config` after confirming with the user |
| View logs | Press `⌃O` or run `paw logs` |
| Help | Press `⌃/` for keyboard shortcuts |

**For detailed instructions, read `.paw/HELP-FOR-PAW.md`**
