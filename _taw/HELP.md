# TAW (Tmux + Agent + Worktree)

## Commands
  .taw/new-task    Create a new task (opens $EDITOR)

## Agent Slash Commands
  /pr           Create PR and open in browser
  /done         Cleanup task (worktree, branch, window)

## Navigation
  ⌥←→           Move between panes
  ^⌥←→          Move between windows
  ^q            Detach from session
